Root Viewing¶

Once found the roots are returned in the form of a RootResult object.

class cxroots.RootResult.RootResult(roots, multiplicities, originalContour)[source]¶

A container to hold roots and multiplicities.

roots¶

list – List of roots

multiplicities¶

list – List of multiplicities where the ith element of the list is the multiplicity of the ith element of roots.

originalContour¶

Contour – The contour bounding the region in which the roots were found.

show(saveFile=None)[source]¶

Plot the roots on the complex plane.

Parameters:saveFile (str, optional) – If provided the plot of the roots will be saved with file name saveFile instead of being shown.

Example

from cxroots import Circle
C = Circle(0, 2)
f = lambda z: z**6 + z**3
r = C.roots(f)
r.show()

from cxroots import Circle
C = Circle(0, 2)

r = C.roots(f)
print(r)
